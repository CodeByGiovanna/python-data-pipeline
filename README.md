# Python Data Pipeline

Projeto desenvolvido para a disciplina **Python para Dados**, demonstrando um pipeline completo de coleta, processamento, armazenamento e análise de dados utilizando Python.

## Objetivos

- Realizar web scraping de uma página HTML.
- Extrair e estruturar os dados coletados.
- Exportar os dados para os formatos CSV, JSON e Excel.
- Manipular dados utilizando Pandas.
- Persistir dados em um banco SQLite com SQLAlchemy.
- Implementar tratamento de exceções para tornar o pipeline robusto.

## Tecnologias utilizadas

- Python 3
- Requests
- BeautifulSoup
- Pandas
- SQLAlchemy
- SQLite
- CSV
- JSON

## Funcionalidades

- Download de páginas HTML.
- Extração de dados com BeautifulSoup.
- Manipulação de listas, dicionários e conjuntos.
- Exportação para CSV, JSON e Excel.
- Análise de dados com Pandas.
- Armazenamento em SQLite utilizando SQLAlchemy.
- Tratamento de exceções para leitura, escrita e consultas SQL.
